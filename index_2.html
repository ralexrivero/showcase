Personal Portfolio

Frontend development project for my personal portfolio. Including HTML, CSS, JavaScript, Bootstrap and FontAwesome is a static project to show my projects and let you know more about me.

The project is hosted on Github Pages, Namecheap DNS with https. The deployment of the project was made with git as a vcs using branches for the development and production.
